# GigATax
Taxes built for creators, streamers, and everyone else the IRS forgot.
## Background
Originally starting as a hackathon project for HackMSA 2026, we built GigATax as a project we thought would be useful for not just others, but also ourselves.

The idea of a software service that can alleviate the pains of taxes and business transaction monitoring for the new generation is what lead us to create GigATax. The Gig is economy is growing faster than ever, with occupations like streaming, editing, and overall independent work being what the new generation is moving more into rather than the corporate route.

## Purpose
With this in mind, we believe that building a platform that can allow users to monitor their expenses and see what can be a deduction/tax write off on their 1099s is something that could be very valuable to many people.
